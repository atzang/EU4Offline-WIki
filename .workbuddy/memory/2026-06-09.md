# 2026-06-09

## 分析 eu4cn.com 爬取站点

- 站点：www.eu4cn.com，《欧陆风云4》中文百科，基于 MediaWiki 1.39.3
- 使用 Liberty 皮肤，主色 #103D8F，归属 ParaWikis 联盟
- 文件规模：wiki/ 含 14,864 个 HTML 页面，images/ 含 23,584 个文件
- 内容分类：游戏机制、国家任务、历史事件、版本更新、DLC、成就、模组等
- 已发现问题：外部链接未离线化、GA/AdSense 残留、PHP Notice 暴露、可成立国家.txt 未正确爬取、相对路径 CSS/JS 在多级目录下可能失效

## 修复离线站点链接和排版

- 编写 fix_links.py：批量处理 wiki/ 下 14864 个 HTML，将绝对 URL 转相对路径、去除 GA/AdSense/PHP Notice、修复图片路径
- 编写 fix_index.py：修复 index.html 侧边栏（替换为静态常用页面列表，含「可成立国家」置顶）、去除 footer 广告和百度统计
- 编写 fix_sidebars.py：批量替换全部 wiki/ 页面侧边栏为静态常用页面列表
- 可成立国家.html 已在主页侧边栏置顶，所有 wiki 页面侧边栏同步添加入口

## 添加返回顶部按钮 + 侧边栏返回首页

- 编写 add_back_to_top_and_fix_sidebar.py：批量为 index.html 和全部 wiki/*.html（共14865个）注入固定定位右下角「返回顶部」按钮
- 按钮样式：蓝色圆形（#103D8F），右下角 fixed 定位，滚动超200px后淡入显示，点击平滑滚回顶部
- 侧边栏末尾新增加粗「返回首页」链接（蓝色加粗，有上分隔线，wiki页面指向../index.html，主页指向./index.html）

## 修复缩略图错误占位符

- 问题：2908 个页面含有 <div class="MediaTransformError">创建缩略图出错：文件丢失</div>，为旗帜图标缺失
- 解决：创建 images/flag_placeholder.svg（20x13px 灰色旗帜占位 SVG），编写 fix_thumbnail_errors.py 批量替换
- 结果：共修改 2883 个 wiki HTML 文件，残留 1 处仅在 meta description 文本中（不影响显示）
