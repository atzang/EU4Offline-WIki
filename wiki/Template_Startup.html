输入：{{startup|Provence|普罗旺斯}}

输出：
1444年，雄心勃勃的安茹的勒内统治着普罗旺斯，他宣传自己不仅是安茹公爵和普罗旺斯伯爵，而且还是阿拉贡、那不勒斯、和耶路撒冷的国王。尽管基督教世界的那些君主们对这些宣称大多不以为意，而贤王勒内已下定决心收回他生来的权利。

普罗旺斯版图支离破碎。安茹位于英格兰与法兰西冲突的中心地带，同时被两国所觊觎，很可能遭到未来战争的蹂躏。在莱茵兰的普罗旺斯领土分为直属的巴尔公爵领和附庸的洛林公爵领。如果普罗旺斯干预莱茵兰诸侯的事务，会有干犯神圣罗马帝国皇帝和强大的勃艮第公国的风险。普罗旺斯公爵领本土坐落在地中海北岸，毗邻法兰西、萨伏依和教宗国的阿维尼翁城。雄心勃勃的勒内可能从此地出发入侵阿拉贡，虽说由于帝国的控制力日渐消退而使得意大利可能更具有吸引力。

普罗旺斯一直和最强大的邻国法兰西保持着友好的关系。假如百年战争再爆发进入全面战争，普罗旺斯必须从支持法国利益的角度出发平衡自己的防守需求和扩张计划。

普罗旺斯是一个雄心勃勃的国家，但是在邻国觊觎下，就连保持其领土不被蚕食可能也是个挑战。


本地化代码位于：\localisation\startup_screen_l_english.yml；判断Tag的脚本代码位于：/Europa Universalis IV/customizable_localization/00_start_screen.txt。

截至目前为止，Startup文本在本百科中只用于国家词条，PAGENAME就是国家名称。

更新模板时对本地化的编辑：

  • §Y黄色模板§! 替换为 {{Yellow}}
  • [GetYear] 替换为斜体（因为是变量）的 ''{{{GetYear|1444}}}''
  • [Root.GetName] 替换为参数和魔术字 {{{Root.GetName|{{PAGENAME}}}}}
  • [Root.GetAdjective] 替换为参数和魔术字 {{{Root.GetAdjective|{{PAGENAME}}}}}
  • [Root.GovernmentName] 替换为参数 {{{Root.GovernmentName|<政体名>}}}
  • [Root.Monarch.Dynasty.GetName] 替换为参数 {{{Root.Monarch.Dynasty.GetName|<王朝>}}}
  • [Root.Monarch.GetName] 替换为参数 {{{Root.Monarch.GetName|<君主>}}}
  • [Root.Religion.GetName] 替换为参数 {{{Root.Religion.GetName|<宗教>}}}
  • [Root.Religion.GetCountryModifiers] 替换为参数 {{{Root.Religion.GetCountryModifiers|<宗教修正>}}}
  • [Root.start_religion_intro] 替换为参数 {{{Root.start_religion_intro|<宗教介绍>}}}
  • string 替换为 | string （有个别例外）
  • \n\n 替换为 <br />
  • \n 替换为 <br/>
  • £DLC_xxx£ 需要人工识别，替换为 {{icon| xxx }} （结尾的£大量丢失）
  • 为特殊tag添加中文名判断，也就是要显示Startup的词条名

最后做总体检查。
