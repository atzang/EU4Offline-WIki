< Template:Mission tree
这是Template:Mission tree的模板文档。
当直接浏览时，使用变量可能会失效，请不要zh-hans:使用硬编码来代替;zh-hant:寫死页面名或地址。

该模板用于生成任务树，具体使用方法请见下文。

参数

参数 描述 状态
title 任务树标题，默认值是页面名称。如果是DLC任务建议叫作“<DLC名>任务”，比如“皇帝任务”。 选填
国家 用于替换第二列通用军事任务的“扩展XXX”中的”XXX“，默认值为国家。多数情况下可以不写。 选填
第一列 用于填写第以列任务，不写就是空白 选填
第二列 用于填写第二列任务，不写就是通用军事任务。如果前面没有指定国家，也可以这样写：第二列={{通用任务|军事任务|国家}} 选填
第三列 用于填写第三列任务，不写就是通用外交任务。也可以这样写：第三列={{通用任务|外交任务}} 选填
第四列 用于填写第四列任务，不写就是通用行政任务。也可以这样写：第四列={{通用任务|行政任务}} 选填
第五列 用于填写第五列任务。此列经常用于填写区域通用任务，参见模板文档 Template:通用任务/doc 选填

用法

要综合运用{{Mission_tree}}、{{Mission_icon}}和{{Mission_arrow}}三个模板描绘任务树。当你要画某个任务时，可以按照下列步骤：

  1. 打开官方百科对应的词条，最好能打开游戏和游戏的源代码。源代码和词条的对照表在 Category:任务。
    • 观察这个任务树有多少列专属内容。如果中间三列只有通用任务，就可以忽略不管。
  2. 在要编辑的任务页面中放入{{Mission_tree}}
    • 如果这个任务需要DLC，那么填入title参数。“皇帝任务”“达摩任务”“不列颠尼亚任务”之类的。
    • 如果这个任务的第二列只有通用的军事任务，填入国家参数，之后就可以不填第二列了。就像下面例子中的“阿曼任务”那样。
  3. 接下来根据看到的任务树，开始描画各列的内容。
    • 用{{Mission_icon}}构造任务图标。图标背景图片的文件名可以从官方百科的表格，或者游戏源代码找到。如果文件名不对就到 Category:任务图标 里面查找对应的图标。还没有就找站长。
    • 用{{Mission_arrow}}构造任务箭头。基本上就是看图识字，按照箭头的形状，填入对应的制表符号。

后面四列经常有通用任务。按照现在的模板设计，如果通用任务下面还有专用任务，就需要在前面写入对应的通用任务。比如大明任务这段： 

|第二列 = {{通用任务|军事任务|大明}}
{{Mission arrow|┓}}
{{mission icon|mission_kowtow|征服朝鲜|大明}}
|第五列 = {{通用任务|亚洲}}

可以看到，第三列和第四列只有通用任务，就不用写了。
