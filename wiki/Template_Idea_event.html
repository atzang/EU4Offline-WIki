Template:Idea event

[阅读] [编辑] 模板文档
这篇文档嵌入自Template:Idea event/doc。 (编辑 | 查看历史)

Idea event can be used to diplay idea group events.

目录

  • 1 参数
  • 2 用法
  • 3 举例
    • 3.1 Loss of Stability
  • 4 转换表

参数

参数 描述 默认值 状态
event_name Title of the event. empty 必填
event_text Ingame description text of the event. empty optional, but desired
ig To specify the required idea group of the event. empty 必填
trigger For additional trigger conditions. empty 可选
modifiers To specify modifiers of the base weight of the event. empty 可选
option conditions Set to 'yes' if the options are conditional. no 可选
options The possible effects of the event. Please use {{option}}. empty 必填
version To specify the major patch for which the event was written or updated. empty 必填

As all advisor events share the same trigger conditions, MTTH values and modifiers, and parts of the effects these values are already set by the template.

用法

{{Event
|version = 
|event_name = 
|event_text = <!-- optional, but desired -->
|ig = 
|trigger = <!-- optional -->
|option =
}}

举例

{{Idea event
|version = 1.13
|event_id = test
|event_name = Loss of Stability
|event_text = This strengthening of the privileges of the nobles is severely upsetting the burghers and merchants. This is creating turmoil in our realm, and may prove disastrous in the long run.
|ig = aristocratic
|trigger = * has a [[stability]] of at least 0.
|options =
{{option
|option_text = Well, they are just peasants.
|effect = This country:
* loses {{icon|stability|24px}} {{red|1}} stability.
}}
}}

will display:

ID

Loss of Stability

这条信息可能已不适合当前版本，最后更新于1.13。
