Template:Advisor event

[阅读] [编辑] 模板文档
这篇文档嵌入自Template:Advisor event/doc。 (编辑 | 查看历史)

Advisor event can be used to diplay advisor events.

目录

  • 1 Parameter
  • 2 Usage
  • 3 Example
    • 3.1 $NATURAL_SCIENTIST$

Parameter

Parameter Description Default Status
event_name Title of the event. empty required
event_text Ingame description text of the event. empty optional, but desired
advisor_type To specify the advisor type of the event. empty required
emod To specify the modifier granted in an option. empty optional
reward If “emod” is set then to specify the effects of the modifier else further rewards. empty optional
version To specify the major patch for which the event was written or updated. empty required

As all advisor events share the same trigger conditions, MTTH values and modifiers, and parts of the effects these values are already set by the template.

Usage

{{Advisor event
|version = 
|event_name = 
|event_text = <!-- optional, but desired -->
|advisor_type = 
|emod = <!-- optional -->
|reward = <!-- optional -->
}}

Example

{{Advisor event
|version = 1.19
|event_name = $NATURAL_SCIENTIST$
|event_text = One of our advisors, $NATURAL_SCIENTIST$, has received great support for their efforts in developing astronomical instruments and measuring and fixing the positions of stars. His/Her observations include a comprehensive study of the solar system and accurate positions of more than 777 fixed stars.
|advisor_type = a [[natural scientist]]
|emod = Scientific Evolution
|reward = {{icon|adm tech cost}} {{green|−10%}} administrative technology cost. 
}}

will display:

$NATURAL_SCIENTIST$

这条信息可能已不适合当前版本，最后更新于1.19。

One of our advisors, $NATURAL_SCIENTIST$, has received great support for their efforts in developing astronomical instruments and measuring and fixing the positions of stars.
